# PinchApp — Таймер Китайского Чаепития

## Концепт

Приложение для iOS и macOS, которое сопровождает домашнюю гунфу-церемонию китайского чая.  
Пользователь выбирает сорт чая — приложение ведёт его по проливам: показывает время выдержки, номер пролива, подсказку и запускает круговой таймер.  
Дополнительно: встроенный калькулятор для расчёта граммов листа под объём чайника.

---

## Целевые платформы

- iOS 17+
- macOS 14+ (Catalyst или нативный SwiftUI)

---

## Экраны и структура

### Главный экран (Tea Session)

**Верхний блок — выбор чая**
- Dropdown (Picker) с сортами чая
- Под ним — описание-комментарий к выбранному сорту (курсивом, серый текст)

**Средний блок — таймер**
- Круговой прогресс-индикатор (красный на тёмном фоне)
- Номер пролива: `Пролив N` (крупно)
- Время до конца: цифры внутри круга
- Кнопка **Старт / Пауза**
- Кнопка **Следующий пролив** (активна после завершения таймера или по длинному нажатию)
- Короткая подсказка к текущему проливу (из таблицы Примечание)

**Нижний блок — переключатель**
- `Таймер` / `Калькулятор` (SegmentedControl или кастомный switcher)

### Экран / таб «Калькулятор»
- Поле ввода объёма чайника (мл)
- Результат: сколько грамм листа нужно (стандарт 5–7 г на 100–120 мл)
- Формула: `масса = объём_мл × 6 / 110` (средняя точка диапазона)
- Показывает диапазон: min (5 г/110мл) и max (7 г/110мл)

## Трёхитерационный план разработки

---

### Итерация 1 — MVP: Таймер работает

**Цель:** пользователь может выбрать чай, запустить таймер по проливам и дойти до конца чаепития.

**Задачи:**

1. **Модель данных**
   - Структуры `TeaType`, `Steep` (номер, секунды, примечание)
   - Статический массив всех 6 сортов с полными таблицами проливов из `teas.md`

2. **SessionViewModel**
   - `@Published currentSteepIndex`, `timeRemaining`, `isRunning`
   - Методы: `start()`, `pause()`, `nextSteep()`, `reset()`
   - `Timer.publish` с интервалом 1 сек

3. **UI — главный экран**
   - `TeaPicker` — простой SwiftUI Picker
   - `CircularTimerView` — кастомная `Shape` (Circle + trim) с анимацией
   - Кнопки Старт / Следующий пролив
   - Текст подсказки

4. **Пролив 0**
   - По умолчанию первый пролив — нулевой (промывка)
   - Тумблер «Пропустить промывку» в шапке сессии

**Результат итерации:** рабочий таймер, можно провести полное чаепитие.

---

### Итерация 2 — Калькулятор + Полировка UI

**Цель:** добавить калькулятор граммов, улучшить визуал до финального дизайна, добавить звук/вибрацию.

**Задачи:**

1. **Калькулятор**
   - `CalculatorView` — поле ввода объёма (мл), слайдер или stepper
   - Формула + показ диапазона min/max
   - Плавное переключение `Таймер ↔ Калькулятор` через нижний SegmentedControl

2. **Финальный дизайн**
   - Цветовая схема: чёрный / тёмно-серый / красный
   - Карточки с закруглёнными углами и subtle shadow
   - Анимация появления подсказки (fade + slide)
   - Декоративный элемент (дракон / бамбук) в фоне через `ZStack`

3. **Уведомления в рамках приложения**
   - Звук / haptic feedback при завершении пролива
   - Пульсация кнопки «Следующий пролив» после таймаута

4. **Сохранение состояния**
   - `UserDefaults`: запомнить последний выбранный сорт чая
   - Восстановление выбора при следующем открытии

**Результат итерации:** приложение выглядит финально, калькулятор работает, есть тактильный фидбек.

---

### Итерация 3 — macOS + История + Шлифовка

**Цель:** адаптировать под macOS, добавить историю сессий, подготовить к релизу.

**Задачи:**

1. **macOS адаптация**
   - Проверить и скорректировать layout под Mac (wider canvas)
   - Меню в menu bar: быстрый старт последнего чая
   - Поддержка клавиатурных шорткатов (Space = старт/пауза, → = следующий пролив)

2. **История сессий**
   - Модель `Session` (дата, сорт, количество проливов)
   - Сохранение в SwiftData или CoreData
   - Экран «История» — список последних 20 сессий

3. **Настройки**
   - Кастомное время пролива (пользователь может отредактировать секунды)
   - Опция «всегда держать экран активным» во время сессии (`UIApplication.shared.isIdleTimerDisabled`)

4. **Релизная подготовка**
   - App Icon (дракон на чёрном фоне)
   - Launch Screen
   - Локализация: русский (основной), английский
   - Privacy manifest

**Результат итерации:** приложение готово к публикации в App Store для iOS и macOS.
