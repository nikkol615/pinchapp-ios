### 🟫 1. Шу Пуэр (Черный пуэр)
**Максимум: 10–12 проливов.**
*Качественный Шу пуэр очень плотный и выносливый. Он долго держит «нефтяной» цвет, а затем плавно уходит в древесную сладость.*

### 🟩 2. Шэн Пуэр (Светлый пуэр)
**Максимум: 12–15 проливов.**
*Шэн — рекордсмен по выносливости (особенно со старых деревьев). На поздних проливах горечь уходит, оставляя потрясающее сладкое послевкусие.*

| Номер | Сколько держать | Примечание |
| :--- | :--- | :--- |
| **0** | **3–5 сек** | **Слить!** Промывка |
| 1 | 3–5 сек | Очень быстро, чтобы не ушел в горечь |
| 2 | 5 сек | |
| 3 | 10 сек | |
| 4 | 15 сек | |
| 5 | 20 сек | |
| 6 | 30 сек | |
| 7 | 45 сек | |
| 8 | 1 минута | |
| 9 | 1 мин 20 сек| Вкус становится более мягким и фруктовым |
| 10 | 2 минуты | |
| 11 | 3 минуты | |
| 12 | 4-5 минут | Можно держать и дольше, лист уже не даст горечи |
*(Примечание: премиальные Шэны могут выдержать и 15+ проливов).*

### 🪨 3. Тёмные улуны (Да Хун Пао, Жоу Гуй)
**Максимум: 8–9 проливов.**
*Утёсные улуны отдают много вкуса и эфирных масел в середине чаепития, но «исписываются» быстрее пуэров.*

| Номер | Сколько держать | Примечание |
| :--- | :--- | :--- |
| **0** | **5 сек** | **Слить!** Промывка |
| 1 | 5–10 сек | Легкий «приветственный» настой |
| 2 | 10 сек | Раскрытие огненных и печеных нот |
| 3 | 15 сек | Пик вкуса (самая яркая чашка) |
| 4 | 20 сек | |
| 5 | 30 сек | |
| 6 | 45 сек | |
| 7 | 1 минута | Терпкость спадает, чай становится водянистым |
| 8 | 1 мин 30 сек| |
| 9 | 2-3 минуты | Завершающий пролив |

### 🔴 5. Красный чай (Дянь Хун, Цзинь Цзюнь Мэй)
**Максимум: 6–7 проливов.**
*Красный чай экстрагируется очень активно. Он дает яркий, мощный вкус в начале, но его ресурс исчерпывается быстрее всего.*

| Номер | Сколько держать | Примечание |
| :--- | :--- | :--- |
| **0** | **3 сек** | **Слить!** Моментальная промывка |
| 1 | 5 сек | |
| 2 | 5 сек | Пик плотности и терпкости |
| 3 | 10 сек | |
| 4 | 15 сек | |
| 5 | 25 сек | |
| 6 | 45 сек | Вкус заметно слабеет, уходит в медовую воду |
| 7 | 1.5 - 2 мин | Дальше заваривать смысла обычно нет |

### ⚪ 6. Белый чай (Бай Му Дань, Шоу Мэй)
**Максимум: 9–10 проливов.**
*Белый чай — самый слабоферментированный. Лист целый, часто с пушком. Он отдает вкус нехотя, плавно, поэтому время выдержки здесь растет быстрее, а ресурс чая довольно большой.*
